## [1.0.0] - 2026-03-01

### Added
- Initial release of BotanyPots - Cobblemon Integrations datapack
- 68 berry recipes (Aguav, Apicot, Aspear, Babiri, Belue, Bluk, Charti, Cheri, Chesto, Chilan, Chople, Coba, Colbur, Cornn, Custap, Durin, Enigma, Figy, Ganlon, Grepa, Haban, Hondew, Hopo, Iapapa, Jaboca, Kasib, Kebia, Kee, Kelpsy, Lansat, Leppa, Liechi, Lum, Mago, Magost, Maranga, Micle, Nanab, Nomel, Occa, Oran, Pamtre, Passho, Payapa, Pecha, Persim, Petaya, Pinap, Pomeg, Qualot, Rabuta, Rawst, Razz, Rindo, Roseli, Rowap, Salac, Shuca, Sitrus, Spelon, Starf, Tamato, Tanga, Touga, Wacan, Watmel, Wepear, Wiki, Yache)
- 7 apricorn recipes (Black, Blue, Green, Pink, Red, White, Yellow)
- 6 mint recipes (Blue, Cyan, Green, Pink, Red, White)
- 8 other Cobblemon plant recipes (Big Root, Galarica Twig, Hearty Grains, Medicinal Leek, Pep-Up Flower, Revival Herb, Saccharine Sapling, Vivichoke)
- 1 MegaShowdown addon recipe (Max Mushroom)
- Complete documentation (README in English and Italian)
- MIT License
- .gitignore file

### Features
- All crops use dirt-based soil tag
- Balanced growth times (1200 ticks = 60 seconds)
- Automatic item drops with configurable chances
- Load conditions via Bookshelf mod integration
