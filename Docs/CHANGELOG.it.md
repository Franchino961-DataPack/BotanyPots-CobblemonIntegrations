## [1.0.0] - 2026-03-01

### Aggiunto

- 68 ricette per bacche (Aguav, Apicot, Aspear, Babiri, Belue, Bluk, Charti, Cheri, Chesto, Chilan, Chople, Coba, Colbur, Cornn, Custap, Durin, Enigma, Figy, Ganlon, Grepa, Haban, Hondew, Hopo, Iapapa, Jaboca, Kasib, Kebia, Kee, Kelpsy, Lansat, Leppa, Liechi, Lum, Mago, Magost, Maranga, Micle, Nanab, Nomel, Occa, Oran, Pamtre, Passho, Payapa, Pecha, Persim, Petaya, Pinap, Pomeg, Qualot, Rabuta, Rawst, Razz, Rindo, Roseli, Rowap, Salac, Shuca, Sitrus, Spelon, Starf, Tamato, Tanga, Touga, Wacan, Watmel, Wepear, Wiki, Yache)
- 7 ricette per albicocche (Nera, Blu, Verde, Rosa, Rossa, Bianca, Gialla)
- 6 ricette per minte (Blu, Ciano, Verde, Rosa, Rossa, Bianca)
- 8 ricette per altre piante Cobblemon (Big Root, Galarica Twig, Hearty Grains, Medicinal Leek, Pep-Up Flower, Revival Herb, Saccharine Sapling, Vivichoke)
- 1 ricetta addon MegaShowdown (Max Mushroom)
- Documentazione completa (README in inglese e italiano)
- Licenza MIT
- File .gitignore

### Funzionalità
- Tutte le colture utilizzano il tag del terreno a base di terra
- Tempi di crescita bilanciati (1200 tick = 60 secondi)
- Drop automatici con probabilità configurabili
- Condizioni di caricamento tramite integrazione mod Bookshelf

## Cronologia delle Versioni

### Come Leggere i Numeri di Versione
- **Maggiore.Minore.Patch** (es. 1.0.0)
  - **Maggiore**: Modifiche incompatibili con versioni precedenti o aggiunta di funzionalità principali
  - **Minore**: Nuove funzionalità, compatibili con versioni precedenti
  - **Patch**: Correzioni di bug e piccoli miglioramenti
